How To Install

  • Download the installer for your OS.
  • Open it and finish setup.
  • Launch Dragon Speedrun.
  • Enter your player name and play.

System Requirements

  • Windows 10+, macOS 12+, or Ubuntu 20.04+.
  • 64-bit CPU.
  • At least 4 GB RAM.
  • Internet connection for multiplayer.

About

A multiplayer dragon-hunt game client built with TypeScript + Electron and distributed through GitHub Releases.
